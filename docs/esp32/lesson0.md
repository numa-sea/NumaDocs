# Lesson0 : Arduino IDEをいれよう

## 1. Arduino IDEとは
- マイコンを動かすためのソフトウェアを開発するために作られた総合開発環境

- ArduinoはC++ベースのプログラミング言語

- 様々なマイコンやコンデンサーを簡単に使用できる
